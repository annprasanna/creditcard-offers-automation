# Credit Card Offers Automation Framework

This is a **full end-to-end test automation framework** for a banking domain credit card offers campaign.  
It includes API testing, UI testing, Cucumber BDD, TestNG, SQL Server validations, and CI/CD with Jenkins.

---

## Features

- API Testing with **RestAssured**
- UI Testing with **Selenium**
- BDD tests with **Cucumber**
- Database validation with **SQL Server**
- CI/CD pipeline using **Jenkins**
- Mock services using **WireMock**
- Git version control integration
